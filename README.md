**Authorization Flow** & **Privileged Gateway Intents** <br/>

**USERNAME** <br/>
`라마봇#5334` <br/> <br/>
**명령어** <br/>
`라마 도움말` 도움말 임베드 출력 <br/>
`라마 날씨` 네이버 날씨 크롤링 출력 (온도, 날씨, 미세먼지) <br/>

openai를 사용한 ai학습 채팅 https://syerco0.com/33 <br/>
